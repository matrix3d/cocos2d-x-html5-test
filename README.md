cocos2d-x-html5-test
====================


cocos2d x 可以用的语言
c++
lua
js

c++ 
优势，稳定 速度快
缺点，编译慢，不利于快速开发
开发工具 vs2012

lua js 属于脚本 
优点 只需编译一次c++程序，更改脚本后无需编译即可运行

cocos2dx html5（推荐） 属于 cocos2dx的一个分支。
优点 可以在浏览器中直接运行。把js代码copy到 cocos2dx的js工程里，即可在手机中运行。

开发工具推荐  idea 谷歌搜索下载
